# Doptex Skills — Gestão de Competências

PWA para gestão de competências e desenvolvimento de colaboradores da Doptex Malhas.

## Funcionalidades

- Matriz de Competências — Heatmap interativo com avaliação por célula
- Provas de Promoção — Integração com Google Forms, aprovação automática >= 80%
- Avaliação de Desempenho — Notas mensais com gráfico de evolução
- Controle de Assiduidade — Registro mensal de faltas e atrasos
- IDC (Índice de Desenvolvimento) — Score composto dos 4 pilares com pesos configuráveis
- Dashboard Gestor — Ranking, radar da equipe, distribuição IDC
- Dashboard Colaborador — Visão pessoal dos 4 pilares + provas pendentes
- 3 perfis — Admin, Gestor, Colaborador
- PWA — Instalável, offline, push notifications

## Stack

Next.js 15 + TypeScript + Tailwind CSS + Firebase (Auth + Firestore) + Vercel
